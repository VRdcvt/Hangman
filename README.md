# Hangman
Задание
Создайте консольную версию игры «Виселица».
Важно:
■ Слово выбирается компьютером из списка слов.
■ Список слов находится в файле в обычном или зашифрованном виде .
■ По завершении игры на экран выводится статистика игры:
• количество времени;
• количество попыток;
• искомое слово;
• буквы игрока.
